# Technologies Used

Java 17 (or your version)
Maven
JUnit (for tests)
VS Code (development environment)


# How to Build & Run
1. Build the project
mvn clean install

2. Run the project
If it’s a normal Java project:
mvn exec:java

If it’s a Spring Boot project:
mvn spring-boot:run


# Running Tests
mvn test

# Packaging
Create a runnable JAR:
mvn package

Output will be under:
/target
